## 📌 About the Project

This repository contains my educational article aimed at simplifying the relationship between **AI, ML, DL, and GenAI** for students, beginners, and anyone exploring the world of **Data Science and Artificial Intelligence**.

The goal of this project is to make complex AI concepts easier to understand through relatable explanations and practical examples.

- Machine Learning (ML)  
- Deep Learning (DL)  
- Generative AI (GenAI)  
- Real-world applications and examples
